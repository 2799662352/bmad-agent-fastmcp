## 🚀 使用方式

### 快速开始
```bash
# 克隆仓库
git clone https://github.com/2799662352/bmad-agent-fastmcp.git
cd bmad-agent-fastmcp

# 安装依赖
pip install -r requirements.txt

# 配置环境变量
cp .env.example .env
# 编辑 .env 文件设置 API 密钥（可选）

# 运行核心版本
python bmad_agent_mcp_core.py
```
